# Aagontuk Render Setup

Build: `npm install`
Start: `npm start`

Required variables:
- NODE_ENV=production
- NODE_VERSION=20.11.1
- APP_URL=https://aagontuk.onrender.com
- ALLOWED_ORIGIN=https://aagontuk.onrender.com
- ADMIN_EMAIL=n.i.farhan44@gmail.com
- GOOGLE_CLIENT_ID=...
- GOOGLE_CLIENT_SECRET=...

Bot fallback and AI admin layout editor are removed. Gemini keys are not needed.

Google redirect URI: `https://aagontuk.onrender.com/api/auth/google/callback`
